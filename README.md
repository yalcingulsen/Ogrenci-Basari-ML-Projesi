# Öğrenci Başarı Tahmin Sistemi

## Proje Hakkında

Bu proje, Bulut Bilişim dersi kapsamında Proje 3: Akıllı Veri Analitiği ve Makine Öğrenmesi Uygulaması için geliştirilmiştir.

Projenin amacı, öğrencilerin çalışma saati, derse devam oranı, önceki not ortalaması ve ödev puanlarına göre dersi geçip geçmeyeceklerini tahmin eden bir makine öğrenmesi modeli oluşturmaktır.

## Kullanılan Teknolojiler

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Joblib
* Random Forest Classifier

## Proje Özellikleri

* Veri seti oluşturma
* Veri analizi
* Makine öğrenmesi modeli eğitme
* Model doğruluk oranı hesaplama
* Eğitilen modeli kaydetme
* Kullanıcıdan veri alarak canlı tahmin yapma
* Grafiksel analiz oluşturma

## Veri Seti

Veri setinde aşağıdaki değişkenler kullanılmıştır:

* study_hours: Haftalık çalışma saati
* attendance: Derse devam oranı
* previous_grade: Önceki not ortalaması
* assignments_score: Ödev puanı
* pass: Geçme durumu

## Elde Edilen Sonuçlar

Model, verilen öğrenci bilgilerine göre öğrencinin dersi geçip geçmeyeceğini tahmin etmektedir. Proje kapsamında model eğitilmiş, test edilmiş ve örnek öğrenci verisiyle tahmin yapılmıştır.

## Sonuç

Bu proje ile veri analizi, makine öğrenmesi modeli geliştirme, tahminleme ve sonuçların görselleştirilmesi süreçleri başarıyla gerçekleştirilmiştir.
